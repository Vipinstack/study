Definition of call, apply, and bind in JavaScript

call:

A method that invokes a function immediately with a specified this context and arguments passed individually. Key Point: Executes the function immediately with given arguments.

apply:

A method that invokes a function immediately with a specified this context and arguments passed as an array. Key Point: Similar to call, but arguments are passed as an array.

bind:

A method that returns a new function with a specified this context and optionally preset arguments, without invoking it immediately. Key Point: Returns a new function that can be invoked later.
